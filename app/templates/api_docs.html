API Reference

Complete documentation for the GCP DevOps Dashboard API

Base URL
https://devops-google-xxxxx.run.app
GET /api/health Public

Health check endpoint for monitoring and uptime checks. Returns the service status and version.

Response
{
  "status": "healthy",
  "service": "devops-google",
  "timestamp": "2024-01-15T10:30:00",
  "version": "1.0.0"
}
GET /api/status Public

Returns the API status and authentication state.

Response
{
  "message": "Hello from Google Cloud Run!",
  "status": "success",
  "authenticated": false
}
POST /login Public

Authenticate a user and create a session.

Request Body
{
  "email": "user@example.com",
  "password": "your-password"
}
Response
Redirects to /dashboard on success
Returns error message on failure
POST /register Public

Create a new user account.

Request Body
{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "min-6-chars",
  "confirm_password": "min-6-chars"
}
GET /api/users Auth Required

Get a list of all registered users. Requires authentication.

Response
{
  "users": [
    {
      "id": 1,
      "name": "John Doe",
      "email": "john@example.com",
      "created_at": "2024-01-15T10:00:00",
      "last_login": "2024-01-15T12:30:00"
    }
  ]
}
Error Response (401)
{
  "error": "Unauthorized"
}
POST /logout Auth Required

End the current user session and redirect to login.
